# Sentiment Analysis

Source code of paper "Enhanced Sentiment Classification on Large-Scale Indonesian Product Reviews: Beyond Traditional N-Gram Approaches"

This original research used dataset from previous research, can be accesed here https://drive.google.com/drive/folders/1QpwRiNdTGPJj9ipoZnPE6N7GLPY2iIV1?usp=sharing..

The research stages when training and testing are carried out with Python using scikit-learn library.
